## 실행방법

1. Clone & Install packages
    
    ```bash
    git clone https://github.com/ZB-Online/Peach-in.git
    
    cd Peach-in
    
    git pull origin main
    
    npm i
    ```
    
2. Run app
    
    ```bash
    npm run start
    ```
